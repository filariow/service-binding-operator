== Release to Quay

1. Login to Quay with your credentials
+
[source,bash]
----
docker login quay.io
----

2. Export needed variables
+
[source,bash]
----
export OPERATOR_REPO_REF=quay.io/<TARGET_REPOSITORY>/service-binding-operator
----

3. Build the image
+
[source,bash]
----
make image
----

4. Push the image
+
[source,bash]
----
make push-image
----
